# ProductMax AI Swarm

**Self-learning, adaptive multi-agent AI team that fuses Design Thinking + Scrum to turn any product idea or RFP into a fully buildable product specification.**

## What It Does
- Takes a product idea description **or** RFP link as input
- Automatically decomposes it into scopes/epics/features
- Generates detailed user stories with acceptance criteria
- Adds innovative features and non-functional requirements
- Iterates through hybrid Design Thinking + Scrum cycles until the product is **ready to build**
- Self-learns and improves with every run

## Agent Team
- **Empathizer Agent** (Design Thinking: Empathize + Define) — user personas, pain points, problem statements
- **Ideator Agent** (Design Thinking: Ideate) — brainstorms features and innovative additions
- **Product Owner Agent** (Scrum) — value maximization, prioritization, Product Goal
- **Story Decomposer Agent** — breaks down into epics, features, and INVEST user stories
- **Validator / Tester Agent** — feasibility checks, prototypes, acceptance criteria, risks
- **Scrum Facilitator Agent** — orchestrates sprints, syncs, reviews, retrospectives
- **Meta-Learning Agent** — runs retrospectives, updates memory, evolves the swarm

## Workflow (Innovation Sprints)
1. Input ingestion & empathy mapping
2. Problem definition & ideation
3. Sprint Planning & backlog creation
4. Decomposition, story writing, feature addition
5. Validation, prototyping, feasibility review
6. Retrospective & adaptation (Meta-Learning Agent improves the system)

Repeat until the output meets "buildable" criteria (complete prioritized backlog, stories with AC, architecture, tech stack, MVP definition, risks).

## Self-Learning
- Shared memory/knowledge base of past products
- Retrospective-driven prompt and process improvement
- Evolutionary refinement over multiple runs
